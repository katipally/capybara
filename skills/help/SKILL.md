# Capybara Help

Render this card.

```
 capybara — calm senior-dev mode. funny name, professional work.

 LEVELS   /capybara low | medium | high | off      ("stop capybara" = off)
          low = nudges · medium = default · high = aggressive

 SKILLS   /capybara:plan    clarify before code: ASCII + questions + edge cases
          /capybara:review  review the diff against the 6 pillars
          /capybara:clean   hygiene pass: dead code, stale comments, sanitize
          /capybara:done    verify-before-done gate (checklist + run the check)
          /capybara:help    this card

 PILLARS  CLARIFY · LEAN · OPTIMAL · ECONOMY · COMPLETE · HYGIENE
          applied proportionally — trivial asks get the rules, not ceremony.
```
